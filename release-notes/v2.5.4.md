# 贞贞的无限画布 v2.5.4

本版新增 Happy Horse 视频、Seed Audio 音频和完整 3D 表情白模工作流，并继续通过 GitHub Release 提供 Electron 自动更新。

## 新增

- 视频节点新增 `Happy Horse`：支持 `happyhorse-1.1-t2v`、`happyhorse-1.1-i2v`、`happyhorse-1.1-r2v`，覆盖文生视频、单图生视频和 1-9 张多参考图生视频。
- 音频节点新增 `Seed Audio`：支持 `doubao-seed-audio-1.0`、WAV/MP3/PCM/OGG Opus、采样率及语速/音量/音高控制，并保持原 Suno 功能不变。
- 新增 3D 表情编辑节点：内置 MIT 许可 ICT FaceKit 中性人类白模、52 表情通道、照片比例校准、模型体检、相机灯光、透明背景和 1K-4K 图片输出。
- 画布教程新增第十六弹 Bilibili / YouTube 入口。

## 优化

- `api.seedance.nz` 统一显示为“贞贞的平价AI工坊（国内）”，`ai.t8star.org` 统一显示为“贞贞的AI工坊（海外）”。
- Happy Horse 与 Seed Audio 固定使用国内平价工坊 API Key，独立路由不会覆盖 Seedance、Seedream、Suno 或既有视频模型。
- 视频和音频任务完成后继续立即转存 T8 本地 output，避免临时 URL 失效。

## 验证

- 四个新模型均已通过真实 API 生成与 FFprobe 校验。
- 3D 白模 GLB、52 morph target、许可证和输出尺寸均有自动化测试。
- Electron 正式包继续强制检查本地私有扩展 sidecar、运行时归档、FFmpeg、Photoshop/Figma/Chrome 插件和自动更新资产，任一缺失即阻断发布。

> 安装包可覆盖旧版本；应用内“检查更新”会读取本 Release 的 `latest.yml`。
