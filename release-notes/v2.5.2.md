# 贞贞的无限画布 v2.5.2

本次为 Electron 自动更新版本，聚焦植物大战僵尸主题的沉浸游玩体验，并同步最新教程公告。

## 更新内容

- 庭院守卫 HUD 新增沉浸大屏按钮，可在大屏中继续当前战局。
- 大屏模式使用 Portal 固定到视口，按 1280x720 战场自适应缩放，不重载 Phaser 场景。
- 支持 Esc 退出：若关卡 / 升级 / 图鉴面板打开，会先关闭面板，再退出大屏。
- 大屏状态会锁定页面滚动并在退出或切换主题时自动清理。
- 顶部“画布教程”新增教程第十五弹，包含 Bilibili 与 YouTube 链接。
- Electron 打包继续校验 Photoshop UXP 插件随包分发。

## 验证

- 庭院守卫引擎 / 主题 / 路由 / 教程公告回归测试
- TypeScript 类型检查与生产构建
- Electron NSIS 打包、post-build 校验与 GitHub Release 自动更新资产回读
