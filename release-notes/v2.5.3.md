# T8 Penguin Canvas v2.5.3

## 新增

- RunningHub 国内站与海外站使用独立 API Key、固定域名和站点选择，旧配置继续默认国内站。
- 贞贞 SD2 `api.seedance.nz` 支持 Seedance 2.0 主力模型，以及 Seedream V5 Pro 文生图和图像编辑。
- ComfyUI 应用支持图片、视频、音频、文本多类型输出，并通过一个聚合输出节点继续下游。
- Chrome 网页素材扩展新增 Popup 与 Side Panel，可扫描图片、背景图、`srcset`、Canvas，支持筛选、视口截图和批量导入。
- 顶部新增“API获取”，集中提供贞贞与 RunningHub 国内/海外入口和服务差异说明。

## 修复与安全

- RunningHub 任务提交后记住实际站点与 Key，轮询、取消不会串站；只有明确鉴权或应用/任务不存在时才尝试另一站。
- 国内与海外相同 WebApp ID 的 RH 工具可同时保存，不再互相覆盖。
- Electron 正式发布改为 fail-closed：缺少必需私有前端、加密后端或 post-build 钩子时立即失败，不再生成不完整用户包。
- 网页远程素材下载增加 DNS、重定向、私网地址、尺寸和内容类型安全检查。

## 发布边界

- 公开仓库不包含私有业务源码、密钥、测试凭据、飞书多维表格实验、VibeX 本地子项目或本机缓存。
- Photoshop、Figma 与 Chrome 插件继续随 Electron 安装包分发。
