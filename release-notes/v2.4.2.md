# 贞贞的无限画布 v2.4.2

本次 Electron 自动更新版新增“批量打标”节点，面向训练素材整理和素材库治理：图像、视频可以批量进入队列，调用贞贞 LLM 独立 Key或扩展 API 视觉模型生成 TAG、自然语言描述、短句 caption 或标准 JSON，并保存为训练集常用的 sidecar 文件。

## 主要更新

- 工具箱新增“批量打标”节点，支持本地多文件、文件夹、拖拽和上游 image / video 素材收集。
- 打标模式覆盖 TAG、自然语言描述、短句 caption、标准 JSON 和自定义 Prompt。
- 输出保存到 `output/batch-tags`，按原素材文件名前缀生成 `.txt` / `.json`，默认不覆盖同名文件，用户可显式开启覆盖。
- 支持并发 1-4、失败继续、重试失败项、本地停止队列、打开输出目录，并把结果写入节点 `outputText` 与 metadata。
- 新增后端 `/api/batch-tags/tag`，复用现有 LLM / Vision provider 适配层；视频默认走关键帧抽帧，同时保留 URL 和 Base64 高级模式。
- ModelScope 扩展 API 推荐 `Qwen/Qwen3-VL-235B-A22B-Instruct`，误选文本 Qwen3 235B 做视觉打标时会自动切到 VL 模型。
- Electron 打包后置检查新增 `backend-enc/routes/batchTags.t8c`，防止桌面端用户包漏带批量打标后端路由。
