# 贞贞的无限画布 v2.5.5

本版新增 Wan 2.7 Spicy 图生视频与 Dola Seedream 5.0 Pro 海外图像模型，并继续通过 GitHub Release 提供 Electron 自动更新。

## 新增

- 视频节点新增 `Wan` TAB，模型为 `wan-2.7-spicy-i2v`。
- Wan 支持一张首帧图、2-15 秒、720p/1080p、可选提示词、负面提示词、公开音频 URL、提示词扩写和随机/固定 seed。
- 图像节点 `Seedream` 在 `api.seedance.nz` 来源下新增 `Dola Seedream 5.0 Pro（海外模型）`。
- 海外模型按参考图自动选择 `dola-seedream-5.0-pro-t2i` 或 `dola-seedream-5.0-pro-i2i`。

## 兼容与安全

- 旧画布和未选择海外模型的画布继续默认使用国内 `seedream-v5-pro-t2i/i2i`。
- Wan、Dola Seedream、Happy Horse、Seedance、Seed Audio 与原 Seedream 使用隔离的模型和路由逻辑。
- 本地参考图继续上传为临时公网 URL；生成成功后媒体立即转存 T8 本地 output。
- 临时测试 Key 未写入源码、设置、文档、日志、构建产物或 Git。

## 验证

- `dola-seedream-5.0-pro-t2i`、`dola-seedream-5.0-pro-i2i` 和 `wan-2.7-spicy-i2v` 均已通过真实 API 生成。
- 专项测试、TypeScript 类型检查、公开源码检查、RH 工具箱检查、生产构建和 Electron post-build 均作为正式发布闸门执行。
- Electron 正式包继续强制检查本地私有 sidecar、运行时归档、FFmpeg、Photoshop/Figma/Chrome 插件和自动更新资产，任一缺失即阻断发布。

> 安装包可覆盖旧版本；应用内“检查更新”会读取本 Release 的 `latest.yml`。
