# 贞贞的无限画布 v2.4.0

本次 Electron 自动更新版聚焦扩展 API 图像编辑稳定性、Electron 下分类弹窗兼容性，以及生成任务的快速停止体验。Agnes 图像编辑已按官方文档修正请求格式，提示词库和素材分类不再依赖浏览器 `prompt()`，图像 / 视频 / SD2.0 节点生成中可以停止本地轮询并立即重新发起新任务。

## 主要更新

- Agnes 在扩展 OpenAI 兼容配置下的图像编辑改走 `/v1/images/generations` JSON 请求；参考图放入 `extra_body.image`，`response_format` 放入 `extra_body.response_format`，避免误用 `/v1/images/edits` multipart 导致 503。
- 保留普通 OpenAI 兼容平台的参考图编辑行为：非 Agnes 平台带参考图时继续走 `/v1/images/edits` multipart，无参考图继续走 `/v1/images/generations`。
- 提示词模板库新增 / 重命名分类、素材右键菜单新增模板分类 / 资源分类，以及缺失提示词补录，全部改为 Electron 内联表单，不再调用 `window.prompt()`。
- 图像节点、视频节点和 SD2.0 节点生成中显示停止按钮；停止会取消本地轮询并让旧任务返回失效，方便用户立刻重新提交新的生成请求。远端已经提交的任务仍可能自然完成。
- 版本号按三段 0-9 规则从 `2.3.9` 进位到 `2.4.0`，继续发布 NSIS 安装包、blockmap 与 `latest.yml` 自动更新资产。

- Release 资产包含 NSIS 安装包、blockmap 与 `latest.yml`。
- 公开源码继续排除 `skill.md`、`roadmap.md`、`local-private/**`、`hermes-agent/**`、`comfyui-anima-t8/**`、Electron 缓存目录、`artifacts/**`、`vibex/**`、本地维护者工具源码、ffmpeg / ffprobe 本地 sidecar 和任何本地私有业务实现。
