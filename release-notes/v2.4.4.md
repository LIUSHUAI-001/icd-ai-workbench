# 贞贞的无限画布 v2.4.4

本次 Electron 自动更新版新增“随机路由”工具节点，并补齐上传素材批量视频打散按钮与微信输入法组合输入体验修复。

## 主要更新

- 新增“随机路由”节点：左侧接受任意上游素材，右侧支持 `total_outputs` 1-100 个动态输出口。
- `random_pass_count` 会在每次运行时精确随机命中指定数量的不重复输出口；未命中的输出口后续分支不会进入运行队列。
- 随机路由支持单节点 RUN、批量运行、组运行，也可位于循环器下游；节点会透传文本、图像、视频、音频和 3D 素材字段。
- 上传素材节点在多视频、多图像、多音频合集下保持“打散”按钮常显，不再需要悬停到特定位置才看见。
- 图像等节点使用的富文本 Prompt 输入框补齐 IME 组合输入保护，微信输入法提交中文时会清理前置泄漏拼音，避免多出几个英文字母；搜狗等正常输入路径不受影响。
