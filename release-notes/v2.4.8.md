# 贞贞的无限画布 v2.4.8

本次是 Electron 自动更新发布版，继续收口插件安装、批量打标、视频素材处理和 Photoshop 联动体验。

## 更新重点

- 顶部工具条在“画布教程”左侧新增独立“插件安装”入口，集中说明 Photoshop UXP、Figma Bridge、网页图片反推 Chrome 扩展的安装位置和使用步骤。
- 批量打标节点补齐素材队列单项删除、LLM/Vision 预设模型、自定义模型、ModelScope Qwen3-VL 235B、统一触发词、TXT/JSON 互斥保存和原素材目录写回；JSON 模式按 Ideogram-4 结构化 caption 输出。
- 上传素材 / 输出素材的视频卡片新增能力轨：首尾帧获取、英伟达极速超分、FlashVsr 质量超分，复用 RH 工具箱能力层和持久化工具清单。
- 视频剪辑节点临时隐藏“打开剪辑台”入口，并修复片段选中、时间线预览宽度和浅色 RH 主题按钮样式回归。
- Photoshop 插件升级到 0.1.3，新增启动诊断 `boot.js`、资产分页、搜索输入高度护栏、Manifest v5 label 兼容和 UXP 面板边界约束；Electron 包会继续携带 `resources/tools/photoshop-bridge/plugin`。
- ComfyUI 应用制作工具修复配置项重复展示和删除无效问题；全局输入框继续加强微信输入法组合输入保护。
- 聚合解析运行时归档解压改走内置 Node 解压兜底，降低用户侧 “运行时归档解压失败” 的概率。

## 打包与自动更新

- 安装包：`T8-PenguinCanvas-Setup-2.4.8.exe`
- 自动更新：`latest.yml`
- 发布脚本：`T8_RELEASE_APPROVAL=release-2.4.8 npm run dist:release`

本版发布前覆盖批量打标、RH 工具箱、运行时归档、视频剪辑、Photoshop 插件、插件安装入口、IME 输入、素材操作和 ComfyUI 应用制作器相关回归，并执行类型检查、公开发布边界检查、RH 工具箱清单检查、前端构建、Electron 打包后置检查和 GitHub Release 校验。
