# 贞贞的无限画布 v2.4.7

本次 Electron 自动更新版继续收口 Photoshop 联动体验，修复插件面板紧凑宽度下按钮文字竖排的问题，并补强资产库、生成、PS 发回画布、画布发送到 PS 的 UI 自动化验证。

## 主要更新

- 修复 `T8 Photoshop Link` 面板在 430px 宽度下，资产页 `刷新` 和设置页 `连接` 按钮被挤成竖排的问题。
- 插件按钮统一增加不换行和最小宽度护栏，输入框 / 下拉框在同一行内可正常收缩，避免窄面板下关键操作变形。
- Photoshop 插件 UI QA 扩展为覆盖资产库加载、资产置入 PS、设置页、PS 当前画面上传回 T8、生成页 provider/model 加载、生成结果展示、生成后置入 PS，以及画布发送到 PS 的命令队列置入。
- 单元测试新增插件按钮不换行和紧凑按钮最小宽度断言，防止后续样式改动再次把中文按钮挤成竖排。
