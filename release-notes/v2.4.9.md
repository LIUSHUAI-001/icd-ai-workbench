# 贞贞的无限画布 v2.4.9

本次是 Electron 自动更新维护版，重点修复视频剪辑转场后的音频错位问题，并补齐导演分镜台的新 mini 模型。

## 更新重点

- 视频剪辑在拖动片段、裁剪片段并添加转场后，源视频音频会跟随实际合成时间线一起进入 xfade / acrossfade，不再按旧时间点二次混音。
- 修复转场场景下可能出现的声音重叠、位移、尾段完全无声等问题；独立音频轨仍会叠加到保留的源视频原声上。
- 导演分镜台模型下拉新增 `seedance-2.0-mini`，与 SD2.0 节点保持同一套 Seedance 2.0 模型体验。
- Electron 打包继续携带 `resources/tools/photoshop-bridge/plugin`，并由后置检查确认 Photoshop 插件资源随包分发。

## 打包与自动更新

- 安装包：`T8-PenguinCanvas-Setup-2.4.9.exe`
- 自动更新：`latest.yml`
- 发布脚本：`T8_RELEASE_APPROVAL=release-2.4.9 npm run dist:release`

## 验证

本版发布前覆盖视频剪辑后端合成、真实 ffmpeg 音频回归、视频剪辑前端静态契约、导演分镜台模型选项、Photoshop 插件资源、类型检查、公开发布边界检查、RH 工具箱清单检查、前端构建、Electron 打包后置检查和 GitHub Release 校验。
