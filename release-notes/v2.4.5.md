# 贞贞的无限画布 v2.4.5

本次 Electron 自动更新版新增 Photoshop 本机联动插件与画布桥接，并修复随机路由在透传素材时误生成独立输出素材节点的问题。

## 主要更新

- 新增 `T8 Photoshop Link` UXP 插件，插件目录随 Electron 包一起分发到 `resources\tools\photoshop-bridge\plugin`。
- 后端新增 `/api/photoshop-bridge`：支持 PS 图层 / 文档导出回当前画布、画布图像素材排队发送到 Photoshop、插件资产库浏览、插件内图像生成和参考图编辑。
- 画布发送素材弹窗新增 `发送到 Photoshop`，仅对图像素材启用，并在弹窗内提示 UXP Developer Tool 插件导入路径。
- Photoshop 面板保留 `资产 / 生成 / 设置` 三个页签，不包含 Agent 页；生成页复用 T8 扩展 API 图像模型，支持文生图与图像编辑，成功结果可自动同步回画布。
- Photoshop 与画布的消息队列增加 in-flight、complete、fail 和 lease 重试，避免画布创建节点失败时任务被提前吃掉。
- Electron 后置校验新增 `backend-enc/routes/photoshopBridge.t8c` 与 Photoshop 插件静态文件检查，防止自动更新包漏带 PS 联动能力。
- 随机路由作为透传路由时不再触发通用 auto-output 生成独立“输出素材”节点，并会清理旧版本自动生成的未移动空输出节点。
