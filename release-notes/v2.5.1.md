# 贞贞的无限画布 v2.5.1

本版新增完整的“植物大战僵尸主题 · 庭院守卫”和 Seedream V5 Pro 图像能力，并继续提供 Windows Electron 自动更新安装包。

## 新功能

- 新增第 14 套庭院守卫主题，重新设计草坪、木质藤蔓 HUD、节点、端口、连线、MiniMap、菜单和浅色 / 深色表现。
- 内置按需加载的 Phaser 3 塔防面板，支持 5×9 草坪、阳光经济、8 种植物、8 种入侵者、15 关、昼夜波次、清道器、升级、图鉴与战局恢复。
- 庭院守卫面板位于右上工具栏下方，可单独隐藏；鼠标离开、窗口失焦、拖动画布时自动暂停。
- 庭院守卫主题默认启用 Grasswalk 音乐，拾取阳光、攻击、升级和波次均有主题化反馈。
- 图像节点新增 Seedream V5 Pro 标签，支持文生图、多图编辑、PNG / JPEG、预设尺寸、自定义 `WIDTHxHEIGHT` 和最多 10 张参考图。

## 稳定性与兼容性

- Seedream 使用独立 JSON `/v1/images/generations` 协议，不修改 GPT2、香蕉、Grok 和 MJ 的既有请求分支。
- 自定义尺寸在前后端同时校验；上游可按模型规则将请求尺寸归一到同宽高比的标准分辨率。
- Electron 后置检查新增庭院守卫音乐资源校验，并继续验证 Photoshop UXP 插件完整随包分发。
- 飞书多维表格与其他本地实验功能仍不进入本次公开源码和自动更新包。
