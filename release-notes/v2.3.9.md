# 贞贞的无限画布 v2.3.9

本次 Electron 自动更新版继续收口图像生成与创作者剪辑工作流：香蕉图像节点补齐 Gemini 官方图像模型，OpenAI 兼容扩展支持参考图编辑，资源库支持按分类上传本地资产，完整视频剪辑台进入更稳的宽屏实底体验。

## 主要更新

- 图像节点「香蕉2」新增 `gemini-3.1-flash-lite-image`，「香蕉 Pro」新增 `gemini-3-pro-image`；两个精确模型走 Gemini 官方 `generateContent` JSON 格式，并继续让 `nano-banana-pro` 等旧模型保持原统一格式。
- Gemini 官方图像返回解析新增 `candidates[].content.parts[].inlineData` 支持，同时兼容上游同步转异步任务后的 `data.data[].url` 成功结果；lite 模型固定提交官方支持的 `1K` 图像尺寸。
- OpenAI 兼容扩展新增 `/v1/images/edits`：当图像生成带参考图时自动改走 multipart 编辑接口，保留无参考图时的 `/v1/images/generations`。
- API 设置页的 OpenAI 兼容说明同步补充 `/v1/images/edits`，便于接入 One API、New API 或自建中转。
- 资源库“我的资产”新增本地上传按钮：用户必须先选择具体分类，再把本地图像、视频、音频或全景文件上传入该分类，避免误传到“全部”。
- 完整视频剪辑台继续收口：宽屏实底布局、63 个转场、音频曲线、任务失败/取消提示、快捷键安全态和多主题布局验收一起进入本次自动更新包。
- Electron 打包继续携带 `ffmpeg` / `ffprobe` sidecar 与 `shared/videoTransitions.json`，打包后置检查会验证 ffprobe JSON 探测和 native xfade 转场支持。
