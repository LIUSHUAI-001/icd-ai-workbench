# 贞贞的无限画布 v2.5.0

本次是 Electron 自动更新维护版，重点收口随机路由并发、即梦 CLI 新能力、批量打标体验和 Photoshop 联动队列。

## 更新内容

- 随机路由新增“并发生成 / 顺序生成”模式，默认并发执行本次随机命中的多路下游；旧画布未配置时自动按并发模式处理。
- 即梦 CLI 适配 v1.4.10：`text2image` / `image2image` 支持 `generate_num` 一次生成 1-10 张图，Seedance 2.0 mini、`seedance2.0_vip` 4K 和 Seedance 1.x 新命名兼容同步补齐。
- 批量打标节点顶部新增图图打标器入口，节点内可定位最近打标文件，并可查看全部生成打标文件的绝对路径。
- 画布顶部新增“图图打标器”入口，打开独立介绍卡片后可跳转获取打标与模型训练工具。
- 发送到 Photoshop 改为后端队列确认，画布提示会提醒用户查看 PS 面板置入状态；PS 插件会轮询队列并只在 Photoshop 确认创建图层后上报完成。
- Photoshop 插件设置页区分后端桥接地址与前端画布地址，“打开 T8 画布”不再打开后端 `Cannot GET /` 页面；资产缩略图在 UXP 面板内改为可渲染的数据源。

## 打包说明

- Electron 包继续携带 `resources/tools/photoshop-bridge/plugin`，包含 manifest、HTML、CSS 与 `boot/state/net/ps/app` 脚本。
- 自动更新资产包含 NSIS 安装包、blockmap 和 `latest.yml`。
- `skill.md`、本地私有扩展、缓存目录、运行时 sidecar 和未发布实验文件不进入公开 GitHub 提交。
